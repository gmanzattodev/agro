# 🌱 AgroNova

## 📖 Sobre o Projeto

O **AgroNova** é um projeto desenvolvido com foco em interatividade e experiência do usuário.
A proposta principal foi criar um site com uma interface dinâmica, onde um celular virtual permite controlar e interagir com diferentes funcionalidades da aplicação.

Este projeto foi criado com o objetivo de aprofundar conhecimentos em desenvolvimento front-end, explorando conceitos de estruturação, estilização e interatividade com JavaScript.
